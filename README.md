# Classic Snake Game in Unity
This is a simple implementation of the classic Snake game built using Unity.
I
## How to Play
- Use WASD to move the snake
- Eat the food to grow longer
- Avoid hitting the walls or yourself

## Built With
- Unity Engine
- C#
